# Task 5 — Медиа: R2 + Cloudflare Stream

**Depends on:** 1 (для интеграции), аккаунт CF — независимо
**Goal:** Загрузки из админки летят в R2; тестовое видео проигрывается только по подписанному URL.

## Scope

- R2: приватный бакет, API-токен; `@payloadcms/storage-s3` для коллекции Media (обложки, PDF).
- Cloudflare Stream: включить, залить тестовое видео, `requireSignedURLs: true`, allowed origins = наши домены.
- `lib/video/`: генерация подписанного playback-токена (короткий TTL) на сервере; unit-тест.
- `lib/pdf/`: скелет выдачи PDF через приложение (проверка доступа → stream из R2); вотермарк — заглушка с TODO (полноценно — в фиче F7).
- Env-переменные задокументированы в `.env.example`.

## Acceptance criteria

- Загрузка файла в админке → объект в R2; публичной ссылки нет.
- Тестовое видео: с валидным токеном играет, по прямому URL — нет.
- PDF-роут отдаёт файл только аутентифицированному (пока без покупки — TODO на F7), 403 гостю.

## Out of scope

- Плеер-UI, вотермарк PDF, привязка видео к урокам в вёрстке.
