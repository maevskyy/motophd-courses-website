# Claude Agent Guide

В репозитории **один** гайд для агентов, чтобы две точки входа не разъезжались.
Канонический контент — порядок чтения, инварианты, карта проекта, команды —
живёт в **[AGENTS.md](./AGENTS.md)**.

Claude Code автоматически загружает этот файл, поэтому его единственная задача —
отправить тебя туда:

➡️ **Сначала прочитай [AGENTS.md](./AGENTS.md).** Всё, что нужно агенту, там или по ссылкам оттуда.
